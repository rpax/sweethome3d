Fonctionnalités

Jeks est un tableur JavaTM.
Comme le montre la liste des fonctionnalités suivante, Jeks 1.0 permet surtout l'édition et le calcul de formules dans les cellules d'une feuille de calcul :

  • Edition de formules dans les cellules avec à disposition de nombreux opérateurs et fonctions.
  • Edition de fonctions paramétrables utilisant des expressions définies par l'utilisateur.
  • Vérification des références circulaires dans les formules et mise à jour optimisée des formules utilisant d'autres cellules.
  • Couper / Copier / Coller avec décalage automatique des références de cellule dans les formules.
  • Internationalisation du tableur et de la syntaxe des formules. Version française et anglaise disponibles uniquement pour l'instant.
  • Enregistrer / Ouvrir des fichiers au format Jeks (fichiers .jks).

Syntaxe des formules

Une formule commence par le caractère =, ne fait pas de nuance entre majuscules et minuscules et sa syntaxe supporte les opérateurs suivants :

  • Opérateurs unaires : + -
  • Opérateurs binaires : ^ * / + - >= <= > < = <> (du plus prioritaire au moins prioritaire).
  • Constantes : VRAI FAUX
  • Fonctions (les paramètres optionnels sont en italique) :
    • Fonctions mathématiques :
      • LN (nombre)
      • LOG10 (nombre)
      • EXP (nombre)
      • RACINE (nombre)
      • COS (nombre)
      • SIN (nombre)
      • TAN (nombre)
      • ACOS (nombre)
      • ASIN (nombre)
      • ATAN (nombre)
      • COSH (nombre)
      • SINH (nombre)
      • TANH (nombre)
      • ENT (nombre)
      • ABS (nombre)
      • ALEA ()
      • MOD (nombre;diviseur)
      • FACT (nombre)
      • SOMME (nombre1;nombre2;...)
    • Fonctions logiques :
      • SI (test;valeur_si_vrai;valeur_si_faux)
      • NON (valeur)
      • ET (valeur1;valeur2;...)
      • OU (valeur1;valeur2;...)
      • VRAI ()
      • FAUX ()
    • Fonctions de date et d'heure :
      • DATE (annee;mois;jour)
      • DATEVAL (texte_date)
      • MAINTENANT ()
      • TEMPS (heure;minute;seconde)
      • TEMPSVAL (texte_temps)
      • ANNEE (date)
      • MOIS (date)
      • JOUR (date)
      • JOURSEM (date)
      • HEURE (date)
      • MINUTE (date)
      • SECONDE (date)
    • Fonctions texte :
      • CAR (unicode)
      • TROUVE (texte_trouve;text;indice_de_depart)
      • CODE (texte)

Les valeurs littérales peuvent être des nombres décimaux, entiers ou des chaînes de caractères entre "". Le séparateur décimal est la virgule et le symbole exponentiel la lettre E (par exemple 12,3E2, 15,3, 0,333E-3).
Les formules peuvent contenir des espaces ou des tabulations. Le séparateur de paramètre dans les appels de fonctions est le point-virgule. Des expressions peuvent être parenthésées entre () pour changer la priorité des opérateurs.

Une cellule est référencée par la ou les lettres de sa colonne suivie(s) de l'indice de sa ligne (par exemple A1, AB12,...) ; la lettre de la colonne ou l'indice de la ligne peuvent être préfixées par le caractère $ pour éviter le décalage de la colonne ou de la ligne pendant une opération de copier / coller (par exemple $A3, B$4, $C$1).
Un ensemble de cellules se note avec deux références de cellule séparées par le caractère : (par exemple A1:A3, B2:C5, $C4:$C8).
La référence d'une cellule peut être utilisée n'importe où dans une formule (un ensemble de cellules ne peut être utilisé que dans la fonction SOMME()).

Le type de la valeur renvoyée par une formule dépend du type de ses paramètres et des opérateurs et des fonctions utilisés dans la formule. Jeks gère les types suivants :

  • entiers
  • decimaux
  • chaînes de caractères
  • dates
  • valeurs booléennes

Exemples de formules :

  • =SOMME(A1:A5;C1:C5)
  • =12 + SI (ET (C1 > 5; C2 < 10); 5; 4)
  • =A1^2 + 2*A1*A2 + A2^2

Menus Jeks

Fichier Nouveau Crée une nouvelle feuille de calcul. Si la feuille en cours d'édition a été modifiée une boite de dialogue vous demande d'accepter d'enregistrer les changements.
Ouvrir... Affiche une boite de dialogue de choix de fichier pour sélectionner un fichier de feuille de calcul et l'ouvre dans Jeks. Seuls les fichiers au format Jeks (fichiers .jks) peuvent être lus dans cette version. Le format Jeks est portable et les formules qu'il contient sont traduites automatiquement dans la syntaxe de la langue en cours d'utilisation dans Jeks.
Enregistrer

Enregistre la feuille de calcul en cours dans un fichier. A la première sauvegarde, une boite de dialogue de choix de fichier est affichée pour que vous choisissiez un nom de fichier.

Enregistrer sous...

Affiche une boite de dialogue de choix de fichier pour choisir un nom de fichier et enregistre la feuille de calcul en cours dans ce fichier. Seuls les fichiers au format Jeks (fichiers .jks) peuvent être générés dans cette version. Ce format enregistre toutes les valeurs et les formules des cellules et les éventuelles fonctions que vous avez saisies.
Quitter Quitte Jeks. Si la feuille en cours d'édition a été modifiée une boite de dialogue vous demande d'accepter d'enregistrer les changements.
   
Edition Couper Copie les cellules sélectionnées dans la mémoire tampon de copie et efface les cellules.
Copier Copie les cellules sélectionnées dans la mémoire tampon de copie.
Coller Colle les cellules de la mémoire tampon de copie en commençant à la position courante, en effectuant une opération de décalage des cellules dans les formules collées si nécessaire.
Effacer Efface les cellules sélectionnées.

 

 
Options  Nouvelle fonction...

Affiche une boite de dialogue vous demandant de saisir une nouvelle fonction du type fonction(parametres)=expression utilisant les parametres, où parametres est une liste de paramètres séparés par des point-virgules, que l'expression peut utiliser.
La syntaxe d'une fonction est à peu près la même que la syntaxe des formules : Elle accepte les mêmes opérateurs et les mêmes fonctions mais ne permet pas d'utiliser des références à des cellules (utiliser des paramètres à la place). La fonction SI() est remplacée par l'opérateur de condition SI ALORS SINON qui permet d'écrire des appels récursifs.
Une fois acceptée par le compilateur de Jeks, une fonction utilisateur peut être utilisée dans n'importe quelle formule d'une cellule ou dans une fonction.

Exemples de fonctions :

  • f(x,y)=x^2 + 2*x*y + y^2
  • fact2 (n)=SI n <= 0 ALORS 1 SINON n * fact2(n-1)
